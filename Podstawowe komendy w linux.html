Linux Commands

1. ping

The ping command is used to test the reachability of a host on an Internet Protocol (IP) network. It sends ICMP Echo Request messages to the specified host and waits for an Echo Reply.

2. su

The su command stands for "switch user" and is used to change the current user to another user, typically the superuser (root), after providing the correct password.

3. ipconfig

Although ipconfig is a command used in Windows, its Linux equivalent is ifconfig. The ifconfig command displays the configuration of network interfaces on the system.
